Technical guide for setting up and using AI-EmergencyShield.

Setup Instructions

To run AI-EmergencyShield locally:

  1. Install Ballerina SDK from ballerina.io.
  2. Clone the repository: git clone https://github.com/yourusername/AI-EmergencyShield.git.
  3. Navigate to the backend/ directory and run bal run.
  4. Install Python dependencies: pip install -r requirements.txt.
  5. Open frontend/index.html in a browser.

API Endpoints

Interact with the platform via these Ballerina services:

  • GET /emergency: Retrieve current status and risk level.
  • POST /emergency/signin: Authenticate a user (JSON payload: {username, password}).
  • POST /emergency/signup: Register a new user (JSON payload: {username, email, password}).
